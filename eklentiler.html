YAZILIM ARŞİVİ

AserLand Network altyapısını güçlendirmek ve sunucu mimarisini tamamen özelleştirmek amacıyla Spigot/Paper API (Java) kullanılarak sıfırdan geliştirilen özel eklenti projeleri.

AserCore (Permissions)
Java Spigot API Aktif Geliştirme

AserLand sunucusunun iskeletini tek başına taşıyan, modüler mimariye sahip devasa bir çekirdek (Core) eklentisidir. Piyasada bulunan onlarca farklı eklentiyi (AuthMe, Essentials, LuckPerms, Vault) kullanmak yerine; tüm bu sistemleri tek bir çatıda, AserLand evreninin kurallarına ve Towny sistemlerine %100 entegre olacak şekilde birleştirir.

  • Özel Güvenlik: Şifreleri düz metin yerine SHA-256 algoritmasıyla şifreleyerek veritabanında koruyan entegre Kayıt/Giriş (Auth) sistemi.
  • AserBank (Vault Hook): Dışarıdan bir ekonomi eklentisine ihtiyaç duymadan, Vault köprüsü üzerinden kendi ekonomi altyapısını (para gönderme, bakiye yönetimi) çalıştırır.
  • GUI Tabanlı Yönetim: Karmaşık komutlara gerek kalmadan, yetkililerin oyuncu rütbelerini ve izinlerini özel bir envanter (Menü) üzerinden yönetmesini sağlar.
  • Dinamik Senkronizasyon: Oyuncuların kasaba/ulus (Towny) bilgilerini, rütbelerini ve ping değerlerini PlaceholderAPI aracılığıyla anlık olarak TAB listesine ve sohbete yansıtır.
// Şifreleme Algoritması (SHA-256) private String hashPassword(String password) { try { MessageDigest digest = MessageDigest.getInstance("SHA-256"); byte[] hash = digest.digest(password.getBytes(StandardCharsets.UTF_8)); StringBuilder hexString = new StringBuilder(); for (byte b : hash) { String hex = Integer.toHexString(0xff & b); if (hex.length() == 1) hexString.append('0'); hexString.append(hex); } return hexString.toString(); } catch (NoSuchAlgorithmException e) { e.printStackTrace(); return password; } }
AserPass
Java Ekonomi / İlerleme Pre-Alpha

Klasik "BattlePass" veya "Görev Bileti" mantığının AserLand evrenine uyarlanmış özel bir versiyonudur. Oyuncular, evrenin zorluklarına karşı (hayatta kalma, blok kırma, belirli yaratıkları avlama) tamamladıkları görevlerle kademe atlar ve doğrudan "AserCore" ekonomisine entegre olan nadir ödüller kazanırlar.

  • AserLand Lore Entegrasyonu: Görevler standart oyun mekaniklerinden ziyade, Skarrgard Devrimi ve Aseria Konseyi gibi hikayesel fraksiyonların temalarına göre şekillenir.
  • Dinamik İlerleme Arayüzü: Oyuncuların hangi kademede olduklarını ve hangi ödülleri açtıklarını görebildikleri şık bir oyun içi menü (GUI) barındırır.
  • AserBank Uyumlu: Verilen ekonomik ödüller anında AserCore veri tabanına işlenerek güvenli bir senkronizasyon sağlar.
